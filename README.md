# Sugah Chengalpattu — Integrated Nursing, Paramedic & Assisted-Living Campus

A single self-contained, interactive project dossier (socioeconomic impact summary,
site & survey drawings, campus masterplan and economic impact) for the proposed
campus in Chengalpattu Taluk & District, Tamil Nadu.

**Live site:** published via GitHub Pages (see the repository's Pages settings).

The entire site is one static `index.html` — no build step, all assets inlined.
